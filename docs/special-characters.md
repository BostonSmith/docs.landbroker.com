Filenames are allowed to contain the following special characters:

- Dash (-). You can use dashes to separate month-day-year
- Underscore (\_)

:::warning

All other special characters, including, but not limited to, the following, should NOT be used in filenames:

- Tilde (~)
- Percent (%)
- Asterisk (\*)
- Backslash (\\)
- Angle brackets (< >)
- Slash (/)
- Pipe (|)
- Period (.)
- Number sign (#)
- Ampersand (&)
- Braces ({ })
- Colon (:)
- Question Mark (?)
- Plus sign (+)
- Quotation mark (â€œ)

:::
